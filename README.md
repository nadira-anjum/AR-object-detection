# AR Object Detection System

## Features
- Real-time object detection using COCO-SSD
- AR bounding boxes with labels and confidence
- Clickable AR objects with information panel
- Person activity detection (standing, walking, sitting)
- Class-specific 3D model visualization using Three.js

## Detected Objects
- Person
- Cell phone
- Bottle
- Cup
- Keyboard

## Technologies Used
- React
- TensorFlow.js
- COCO-SSD
- Three.js / React Three Fiber
- HTML Canvas
